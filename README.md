Setup
=====

Please pull from this repo.

In order to run the app, you just need node js, which comes with npm, the node package manager.
http://nodejs.org/

Go to the app's root directory and type

```
node app.js
```

into the terminal.

Then go to

```
localhost:3000
```

in your browser.


For best results, run this in Ubuntu.
